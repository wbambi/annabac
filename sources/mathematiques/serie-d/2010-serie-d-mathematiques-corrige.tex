\textbf{1.} La fonction $x \mapsto -2x + 1 + 2x\ln|x|$ existe pour tout $x < 0$ ; la fonction $x \mapsto (x + 2)e^{-x} - 1$ existe pour tout $x \ge 0$. Donc l'ensemble de définition de $f$ est $\R$.

\textbf{2. a.} \underline{Continuité en 0.} $\lim\limits_{x\to 0^-} f(x) = \lim\limits_{x\to 0^-}(-2x + 1 + 2x\ln|x|) = 1$ ; $\lim\limits_{x\to 0^+} f(x) = \lim\limits_{x\to 0^+}((x+2)e^{-x} - 1) = 1 = f(0)$. Comme $\lim\limits_{x\to 0^-} f(x) = \lim\limits_{x\to 0^+} f(x) = f(0)$, la fonction $f$ est continue en 0.

\underline{Dérivabilité en 0.} $\lim\limits_{x\to 0^-}\dfrac{f(x) - f(0)}{x} = \lim\limits_{x\to 0^-}(-2 + 2\ln|x|) = -\infty$ ; $\lim\limits_{x\to 0^+}\dfrac{f(x) - f(0)}{x} = \lim\limits_{x\to 0^+}\left(e^{-x} + 2\dfrac{e^{-x} - 1}{x}\right) = -1$. Comme $\lim\limits_{x\to 0^-}\dfrac{f(x) - f(0)}{x}$ n'est pas une valeur finie, $f$ n'est pas dérivable en 0.

\textbf{b.} $|x| = -x$ si $x < 0$. La fonction $f$ s'écrit $f(x) = -2x + 1 + 2x\ln(-x)$ si $x < 0$. D'où $\forall x \in ]-\infty\,;0[$, $f'(x) = -2 + 2\ln(-x) + 2x \times \dfrac{-1}{-x} = 2\ln(-x) = g(x)$.
