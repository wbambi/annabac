\textbf{1.} Soit $z = ib$ où $b \in \R$, une solution de l'équation $Z^4 - \sqrt{2}\,Z^3 - 4\sqrt{2}\,Z - 16 = 0$.
\begin{align*}
(ib)^4 - \sqrt{2}(ib)^3 - 4\sqrt{2}(ib) - 16 = 0 &\Longleftrightarrow b^4 - 16 + i\sqrt{2}\,b(b^2 - 4) = 0 \\
&\Longleftrightarrow b^4 - 16 = 0 \text{ et } b(b^2 - 4) = 0 \\
&\Longleftrightarrow b = 2 \text{ ou } b = -2
\end{align*}
Les solutions imaginaires pures sont $z_0 = 2i$ et $z_1 = -2i$. L'équation peut alors se mettre sous la forme :
\[ Z^4 - \sqrt{2}\,Z^3 - 4\sqrt{2}\,Z - 16 = (Z - 2i)(Z + 2i)(Z^2 + cZ + d) = (Z^2 + 4)(Z^2 + cZ + d). \]
Le terme de degré 0 du second membre est $4d$ : $4d = -16$, d'où $d = -4$. Le terme de degré 1 est $4c$ : $4c = -4\sqrt{2}$, d'où $c = -\sqrt{2}$. Il vient : $Z^4 - \sqrt{2}\,Z^3 - 4\sqrt{2}\,Z - 16 = (Z^2 + 4)(Z^2 - \sqrt{2}\,Z - 4)$.

L'équation $Z^2 - \sqrt{2}\,Z - 4 = 0$, de discriminant $\Delta = (-\sqrt{2})^2 - 4(-4) = 18$, admet deux racines $z_2 = 2\sqrt{2}$ et $z_3 = -\sqrt{2}$. Donc les solutions sont : $z_0 = 2i$, $z_1 = -2i$, $z_2 = 2\sqrt{2}$ et $z_3 = -\sqrt{2}$.

\exo{2}{}

\textbf{1.} $\displaystyle I_1 = \int_0^1 x\,e^{-\frac{x^2}{2}}\,dx = \left[-e^{-\frac{x^2}{2}}\right]_0^1 = -e^{-\frac{1}{2}} + 1$.

\textbf{2.} Soit $n \in \N^*$. La fonction $x \mapsto x^n e^{-\frac{x^2}{2}}$ est continue sur $[0\,;1]$. De plus, $\forall x \in [0\,;1]$, $x^n e^{-\frac{x^2}{2}} \ge 0$. Donc $I_n = \displaystyle\int_0^1 x^n e^{-\frac{x^2}{2}}\,dx \ge 0$ pour tout $n \in \N^*$.

\textbf{3.} Soit $n \ge 3$. $I_n = \displaystyle\int_0^1 x^{n-1} \cdot x\,e^{-\frac{x^2}{2}}\,dx$. En posant $u = x^{n-1}$, $v' = x\,e^{-\frac{x^2}{2}}$, donc $u' = (n-1)x^{n-2}$, $v = -e^{-\frac{x^2}{2}}$ :
\[ I_n = \left[-x^{n-1}e^{-\frac{x^2}{2}}\right]_0^1 + (n-1)\int_0^1 x^{n-2}e^{-\frac{x^2}{2}}\,dx = -e^{-\frac{1}{2}} + (n-1)I_{n-2}. \]
D'où $I_n = -e^{-\frac{1}{2}} + (n-1)I_{n-2}$ pour tout entier $n \ge 3$.

\textbf{4.} \underline{Décroissance de $(I_n)$.} Soit $n \in \N^*$.
\[ I_{n+1} - I_n = \int_0^1 x^{n+1}e^{-\frac{x^2}{2}}\,dx - \int_0^1 x^n e^{-\frac{x^2}{2}}\,dx = \int_0^1 x^n(x - 1)e^{-\frac{x^2}{2}}\,dx. \]
La fonction $x \mapsto x^n(x-1)e^{-\frac{x^2}{2}}$ est continue sur $[0\,;1]$ et $\le 0$ sur $[0\,;1]$. Donc $I_{n+1} - I_n \le 0$ : la suite $(I_n)$ est décroissante.

\underline{Convergence.} La suite $(I_n)$ est décroissante (d'après 4.) et minorée par 0 (d'après 2.). Donc elle converge vers une limite $l$.

\textbf{5.} Soit $n \in \N^*$. $\forall x \in [0\,;1]$, $-\dfrac{x^2}{2} \le 0$. Par croissance de l'exponentielle, $0 < e^{-\frac{x^2}{2}} \le 1$. On en déduit $0 \le x^n e^{-\frac{x^2}{2}} \le x^n$. Par passage à l'intégrale :
\[ 0 \le \int_0^1 x^n e^{-\frac{x^2}{2}}\,dx \le \int_0^1 x^n\,dx = \left[\frac{x^{n+1}}{n+1}\right]_0^1 = \frac{1}{n+1}. \]
D'où $0 \le I_n \le \dfrac{1}{n+1}$ pour tout $n \in \N^*$.

\underline{Calcul de $l$.} Par passage à la limite, $0 \le \lim\limits_{n\to+\infty} I_n \le \lim\limits_{n\to+\infty}\dfrac{1}{n+1} = 0$. On en déduit que $l = 0$.
